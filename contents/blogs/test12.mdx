在 Next.js App Router 中：

- Server Components 默认在服务器渲染，不参与客户端水合
- Client Components（标记 "use client"）才会参与水合
- 水合时，React 会将服务器生成的 HTML 与客户端虚拟 DOM 进行匹配（reconcile），然后绑定事件监听器
